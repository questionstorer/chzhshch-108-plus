主材料来自仓库中的 `../108/*-001.md` 到 `../108/*-108.md`，按 UTF-8 读取。原文评论区默认不进入主文；只有作者回复涉及概念澄清时，才可作为注记来源。`W003`、`W004`、`W005` 以及其他非正课文件只收入附录索引，不作为主体系依据。

## 证明口径

本文只在形式条件足够明确时使用“定理”。每个定理后必须紧跟证明。若原文使用“必然”“100%”“当且仅当”等强表述，但依赖市场经验、操作习惯或未形式化的走势选择规则，本文将其降格为经验命题、操作原则或待补假设命题。

特别地，“走势必完美”在本文中不作为无条件市场事实，而作为有限序列、确定性分解规则和递归级别结构下的唯一分解命题。
